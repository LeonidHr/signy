## \#Features
* Sending the form to the mail
* Form validation
* "Show more" functionality
* Spollers 
* Video in popup

## \#Dependencies

#### PHPMailer
For sending the form to the mail

```php
require 'phpmailer/src/Exception.php';
require 'phpmailer/src/PHPMailer.php';
require 'phpmailer/src/SMTP.php';
```
